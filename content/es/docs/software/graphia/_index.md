Una completa colección de plugins de Python diseñados para la aplicación de gráficos [Graph](https://www.padowan.dk/). Este sistema extiende la funcionalidad de Graph añadiendo análisis avanzado de señales, filtrado, importación/exportación de datos, morphing de señales, herramientas de visualización y capacidades de generación de formas de onda.

---

## Indice 

- Instalación
- 1. Análisis
  - Transformada Rápida de Fourier (FFT)
  - Transformada Inversa Rápida de Fourier (IFFT)
- 2. Suavizado
  - Filtro Gaussiano
  - Filtro Mediano Selectivo
- 3. Filtrado
  - Filtro FIR
  - Convolución
- 4. Importación
  - Importador Avanzado de CSV
  - Gestor de Perfiles
- 5. Exportación
  - Exportador de CSV Avanzado
- 6. Morphing
- Morph - Transformar serie
    - Remuestrear
    - Información de la señal
    - Aplicar función
    - Combinación lineal
    - Recortar/Cortar
    - Rellenar segmento
    - [Interpolación espectral](spectral-interpolation/)
- 7. Visualización
  - Dibujar Rectángulo Visible
- 8. Generación de Formas de Onda
  - Generador de Señal Compuesta
  - Generador de Funciones AI
  - Generador de Pulso Gaussiano
  - Generador de Onda Cuadrada
  - Generador de Ruido
  - Muestreador de Funciones
- 9. Utilidades
  - Gestor de Plugins
  - Mostrar Consola
- Autor
- Licencia
